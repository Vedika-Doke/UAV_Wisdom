# UAV_Wisdom

A structured knowledge base built while learning, comparing, and eventually designing UAV systems — flight controllers (FCs), ESCs, firmwares, and the autonomous stack.

This repo is organised by **project phase**. Each phase has its own folder with self-contained notes, comparison matrices, and curated resources.

---

## Phases

| Phase | Focus | Status |
|------:|-------|:------:|
| [Phase 1](./Phase_1) | UAV fundamentals, hardware/firmware survey, comparison matrices (research / market study) | 🟡 in progress |
| Phase 2 | Hardware design: General commercial-grade FC + Lightweight AIO FC+ESC | 🔴 not started |

---

## How to navigate

- Each topic folder has a `README.md` acting as a 1-screen overview + table of contents.
- Comparison files include a Markdown table **and** a parallel CSV in `Phase_1/matrices/` for reuse.
- Every note ends with a **Sources** section so claims can be traced and cited in the final report/PPT.
- Diagrams use [Mermaid](https://mermaid.js.org/) where possible (GitHub renders them natively).
