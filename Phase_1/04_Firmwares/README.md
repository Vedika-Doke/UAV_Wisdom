## TL;DR
- **Betaflight** — FPV acro, no autonomy.
- **INAV** — Betaflight fork with GPS/nav features; great for fixed-wing.
- **ArduPilot** — most mature, widest vehicle support, C++ codebase.
- **PX4** — modular, NuttX RTOS, Apache-2.0 licensed, preferred for commercial products.
