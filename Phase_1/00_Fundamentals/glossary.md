# Glossary

Acronyms and protocols that come up repeatedly in UAV documentation.

| Term | Stands for / meaning | Context |
|------|----------------------|---------|
| FC | Flight Controller | The brain of the drone |
| ESC | Electronic Speed Controller | Drives the brushless motor |
| IMU | Inertial Measurement Unit | Accelerometer + gyroscope |
| BEC | Battery Eliminator Circuit | Steps down LiPo to 5 V / 3.3 V |
| PWM | Pulse Width Modulation | Legacy motor / servo signalling |
| DShot | Digital ESC protocol | Replaces PWM/OneShot — DShot150/300/600/1200 |
| SBUS | Serial protocol from Futaba | RC receiver → FC |
| CRSF | Crossfire serial protocol | TBS / ExpressLRS receivers |
| MSP | MultiWii Serial Protocol | Used by Betaflight/INAV configurators |
| MAVLink | Micro Air Vehicle Link | Messaging protocol used by PX4/Ardupilot/QGC |
| UART | Universal Async Rx/Tx | Serial bus on FCs |
| I²C | Inter-Integrated Circuit | Bus for mag, baro, etc. |
| CAN / DroneCAN | Controller Area Network | Robust bus for smart peripherals |
| VTX | Video Transmitter | Analog/digital FPV video link |
| OSD | On-Screen Display | Telemetry overlay on FPV feed |
| GCS | Ground Control Station | QGroundControl, Mission Planner |
| RTK | Real-Time Kinematic GPS | Centimetre-grade positioning |
