# FPV / Hobbyist Flight Controllers

Cheap, light, F4/F7/H7-class MCUs running **Betaflight** or **INAV**. Optimized for acro/racing, not autonomy.

## Common boards (to study)
- SpeedyBee F405 V4 / V5
- SpeedyBee F7 / H7
- iFlight BLITZ F7
- HGLRC Zeus
- Holybro Kakute H7

## What to extract for each
- MCU (F411 / F405 / F722 / F745 / H743)
- IMU (ICM-42688-P / BMI270 / MPU6000)
- Mounting (20×20, 25.5×25.5, 30.5×30.5)
- UART count
- Onboard BEC + current rating
- Black box (onboard flash vs SD card)
- Barometer present?
- Price range
