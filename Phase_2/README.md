# Phase 2 — Helicopter AIO PCB Design

**Goal:** Design an All-In-One (AIO) Flight Controller + ESC PCB for a **single-rotor helicopter drone**.  
This phase combines NPTEL coursework on drone systems with hands-on PCB design work.

## Design Targets (AIO Helicopter FC+ESC Board)

| Parameter | Target / Notes |
|-----------|---------------|
| Platform | Single-rotor helicopter (swash-plate type) |
| Servo outputs | ≥ 4 (2× cyclic, 1× collective, 1× tail) |
| ESC stage | Single high-current brushless motor (main rotor) |
| RPM sensing | Tachometer / back-EMF input for governor mode |
| Vibration isolation | Soft-mount strategy or damping layer in PCB stack |
| Form factor | AIO (FC + ESC on one board) — compact, low mass |
